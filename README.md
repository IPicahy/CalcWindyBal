## Архитектура и основной функционал
Программа разделена на три логические вкладки, каждая из которых решает свою тактическую задачу. Интерфейс выполнен в темной "Stealth" палитре для снижения нагрузки на глаза.

1. Вкладка «Калькулятор» (Основной рабочий экран)
Предназначена для расчета поправок по конкретной цели с учетом текущих метеоусловий.

 Топография: Позволяет вводить перепад высот либо через дальномер (сразу в метрах выше/ниже), либо по топографической карте (абсолютная высота стрелка и цели). Калькулятор автоматически вычисляет угол места цели (косинус) и корректирует вертикальную поправку.

### Ветер (3 режима ввода):

1. Метеостанция: Прямой ввод данных от корректировщика. Поля "Общая скорость" и "?" созданы для удобства чтения данных сверху вниз. Раздельный ввод бокового (Cross) и прямого (Head) ветра.

2. Вектор ветра: Математический расчет сноса на основе азимута стрельбы, скорости ветра и направления, откуда он дует.

3. Боковой: Быстрый ручной ввод сноса, если значение уже известно.

### Система логов и мульти-целей:

Программа автоматически запоминает рассчитанные цели (добавляя к ним введенное описание).

При изменении ветра достаточно нажать «Рассчитать вынос» — калькулятор пересчитает поправки сразу для всех сохраненных целей в логе под новые метеоусловия.

#### Важно: Если азимут стрельбы меняется более чем на 10 градусов, лог целей автоматически стирается, так как предполагается смена огневой позиции или сектора стрельбы.

2. Вкладка «Авто-Калибровка»
Предназначена для вычисления реального влияния ветра на пулю (настройки параметра wind_factor) прямо в полевых условиях.

### Как пользоваться:

1. Выберите профиль оружия, дистанцию и текущий ветер.

2. Сделайте выстрел строго в центр мишени (без поправок на ветер).

3. Измерьте по сетке прицела, на сколько MIL пулю сдуло по горизонтали в реальности.

4. Введите это значение в блок «ПУЛЮ СДУЛО (Горизонталь)» и укажите направление сноса.

5. Нажмите кнопку вычисления. Скрипт рассчитает новый wind_factor и автоматически обновит файл profiles.json.

3. Вкладка «Редактор оружия»
Позволяет создавать новые или изменять существующие профили винтовок.

### Редактируемые параметры:

v0 (Начальная скорость): Скорость пули на срезе ствола (м/с). Влияет на время полета и, соответственно, на ветровой снос.

k_drag (Аэродинамическое сопротивление): Коэффициент потери скорости. Чем меньше значение, тем медленнее пуля теряет скорость.

wind_factor: Искусственный множитель чувствительности пули к боковому ветру.

#### Примечание: Вертикальное падение на разных дистанциях привязано к LUT-таблицам (scope_table), которые в данной версии зашиты напрямую в код. Чтобы изменить кривую падения пули, необходимо отредактировать словарь DEFAULT_PROFILES в самом скрипте.

Сохранение данных
Программа создает два локальных файла в папке со скриптом:

profiles.json — база данных всех профилей оружия. Загружается при старте.

session_logs.json — временный файл с историей целей текущей сессии.

Оба файла обновляются автоматически.
